# homebrew-ytt

Homebrew tap for [youtube-transcript-tools](https://github.com/nilukush/youtube-transcript).

## Installation

```bash
# Add the tap
brew tap nilukush/ytt

# Install youtube-transcript-tools
brew install youtube-transcript-tools
```

## Usage

After installation, the `ytt` command will be available:

```bash
ytt fetch "https://youtu.be/dQw4w9WgXcQ"
```
